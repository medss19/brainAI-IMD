VGG

The VGG-16 model is a popular convolutional neural network introduced by the Visual Geometry Group (VGG) from the University of Oxford. Known for its simplicity and efficiency, VGG-16 is particularly effective for image classification tasks.

About Model

VGG-16

The VGG-16 architecture consists of:

  • Convolutional layers: The model consists of 13 convolutional layers grouped into five blocks. Each block uses small 3x3 filters, emphasizing deeper networks over wider filters. This choice enhances the ability to extract complex features while keeping computations manageable.


  • Pooling layers: After each block, max-pooling layers (2x2) reduce the spatial dimensions, preserving key features while reducing computational cost.


  • Fully Connected Layers: At the end of the convolutional pipeline, there are two dense layers (each with 4096 units) followed by a softmax layer for classification.


  • Pretrained Weights: VGG-16 often uses weights pretrained on ImageNet, providing a strong baseline for transfer learning tasks.

Training and Validataion sets

Layers

A Flatten layer to reshape the output of the VGG-16 feature extractor.
A Dense layer with 512 units for feature learning. A Dropout layer to reduce overfitting. A Final Dense layer with 4 units (for the number of classes) and softmax activation for classification.
