ResNet-50

The ResNet-50 architecture consists of:

  • Convolutional layers: These layers extract low- and high-level features from the input images.
    1. Input Layer: Shape: (128, 128, 3) (images resized to 128x128 pixels with 3 channels, RGB). Normalization: Input images are typically normalized to have values between 0 and 1 by dividing by 255.
    2. Initial Convolution (Conv1): Conv Layer: A 7x7 convolutional layer with 64 filters, followed by Batch Normalization (BN) and ReLU activation. Max Pooling: A 3x3 max pooling layer with stride 2 is applied after the convolution to reduce the spatial dimensions.


  • Residual blocks: These layers help solve the vanishing gradient problem by using identity shortcuts.
    1. Stage 1: Contains 3 residual blocks. Each block has two convolutions (3x3 kernels) and a skip connection (identity mapping) that helps mitigate the vanishing gradient problem.
    2. Stage 2: Contains 4 residual blocks. Each block has three convolutions (3x3 kernels), with batch normalization and ReLU activation functions applied at each step.
    3. Stage 3: Contains 6 residual blocks. Each block has three convolutions.
    4. Stage 4: Contains 3 residual blocks. Each block has three convolutions.


  • Bottleneck Layers: These ensure that the computations remain efficient even with the increase in depth. Each residual block in the deeper layers is made more efficient by using a 1x1 convolution before the 3x3 convolution to reduce the depth of the intermediate feature maps.


  • Global average pooling: This operation reduces the spatial dimensions, summarizing the features extracted.


  • Dense layer with softmax activation: This produces the final classification output.
    1. Output Shape: (4,), representing the 4 classes (glioma, meningioma, notumor, pituitary).
    2. Activation: Softmax, which outputs probabilities for each of the 4 classes.
